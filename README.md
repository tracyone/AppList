# 为什么

看到github上有关于各种平台上app的推荐，也就是[这货](https://github.com/hzlzh/Best-App)，竟然有4k+的star。我觉得总结还行但是没有加入自己的体验心得，很多软件往往一句“不用解释”或者“你懂的”来敷衍，我觉得这样的推荐和没有推荐差不多。再有很多推荐的软件明显偏向于极客，偏向于小众，国内的一律不说这让小白无所适....

再有推荐了同种用处的不同软件，如果是分付费和免费就算了，但是付费的几种，免费的几种，请问这也叫做**Best App**？最终只会让新手更加的纠结和迷惑。

所以我决定自己搞个，并尽量写出自己的心得。
